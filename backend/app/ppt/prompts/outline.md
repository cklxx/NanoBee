## 核心要求
1. **逻辑严密**：从背景、问题、解决方案到展望，层层递进
2. **数据支撑**：引用参考资料中的数据和案例
3. **简洁有力**：标题8-12字，每条要点15-25字

## 输出格式
请生成一个**JSON格式**的大纲，包含**6-8页**内容。每页包含：
- `title`: 页面标题（8-12字）
- `bullets`: 2-3条核心要点（每条15-25字）

**JSON结构示例：**
```json
{{
  "ppt_outline": [
    {{
      "title": "人工智能发展历程",
      "bullets": [
        "1950年代：图灵测试与AI诞生",
        "1980-2000年：专家系统兴衰",
        "2010年后：深度学习突破"
      ]
    }}
  ]
}}
```

**页面结构参考：**
1. 封面：主题 + 核心价值
2. 背景：行业现状/宏观环境
3. 挑战：核心问题/痛点
4-6. 解决方案：关键方法论（2-3页展开）
7. 展望：趋势/行动建议
8. 封底：总结

**重要：请直接输出JSON，不要添加额外说明文字。**
